This page provides general utility functions for unit conversions, physical constants, and common calculations.

## Unit Conversion
=ee.CONVERT()

ex) Convert values between different units <br/>
`=ee.CONVERT(value, from_unit, to_unit)`

**Temperature Conversion** <br/>
`=ee.CONVERT(100, "C", "F")` → `212` °F  <br/>
`=ee.CONVERT(273.15, "K", "C")` → `0` °C  <br/>

**Pressure Conversion** <br/>
`=ee.CONVERT(1, "bar", "psi")` → `14.504` psi  <br/>
`=ee.CONVERT(101.325, "kPa", "atm")` → `1` atm  <br/>

**Length Conversion** <br/>
`=ee.CONVERT(1, "m", "ft")` → `3.281` ft  <br/>
`=ee.CONVERT(25.4, "mm", "inch")` → `1` inch  <br/>

**Mass Conversion** <br/>
`=ee.CONVERT(1, "kg", "lb")` → `2.205` lb  <br/>
`=ee.CONVERT(1000, "kg", "ton")` → `1` ton  <br/>

**Supported Unit Categories:**
- **Temperature**: C (Celsius), F (Fahrenheit), K (Kelvin)
- **Pressure**: bar, psi, Pa, kPa, MPa, atm
- **Length**: m, cm, mm, km, ft, inch, yard
- **Mass**: kg, g, mg, lb, oz, ton
- **Volume**: m3, L, mL, gal, ft3
- **Energy**: J, kJ, cal, kcal, BTU, kWh

## Physical Constants
=ee.CONSTANT()

ex) Get universal physical constants <br/>
`=ee.CONSTANT(constant_name)`

**Universal Gas Constant** <br/>
`=ee.CONSTANT("R")` → `8.314` J/(mol·K)  <br/>

**Standard Gravity** <br/>
`=ee.CONSTANT("g")` → `9.807` m/s²  <br/>

**Avogadro's Number** <br/>
`=ee.CONSTANT("Na")` → `6.022e23` mol⁻¹  <br/>

**Boltzmann Constant** <br/>
`=ee.CONSTANT("k")` → `1.381e-23` J/K  <br/>

**Planck Constant** <br/>
`=ee.CONSTANT("h")` → `6.626e-34` J·s  <br/>

**Speed of Light** <br/>
`=ee.CONSTANT("c")` → `2.998e8` m/s  <br/>

**Stefan-Boltzmann Constant** <br/>
`=ee.CONSTANT("sigma")` → `5.670e-8` W/(m²·K⁴)  <br/>

**Available Constants:**
- **R**: Universal gas constant (8.314 J/(mol·K))
- **g**: Standard gravity (9.807 m/s²)
- **Na**: Avogadro's number (6.022e23 mol⁻¹)
- **k**: Boltzmann constant (1.381e-23 J/K)
- **h**: Planck constant (6.626e-34 J·s)
- **c**: Speed of light (2.998e8 m/s)
- **sigma**: Stefan-Boltzmann constant (5.670e-8 W/(m²·K⁴))
- **pi**: Pi (3.14159...)
- **e**: Euler's number (2.71828...)
