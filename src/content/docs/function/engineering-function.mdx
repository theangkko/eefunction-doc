## Steam Water - IF97 Steam Table 
=ee.STEAMxxx()

ex) Get Steam Entalpy with (Pressure, Temperature) <br/>
`=ee.STEAMH_PT()`

Input as 2 bara pressure and 150 deg.C Temperature  <br/>
`=ee.STEAMH_PT(2, 150)`    
`=2769.09` kJ/kg Enthalpy of Steam <br/>

## Seawater Properties - MIT Model
=ee.SEAWTR_xxx()

Seawater thermophysical properties based on MIT seawater property library.  
All functions use Temperature (°C), Salinity (g/kg), and Pressure (bar.a) as inputs.

ex) Get Seawater Density with (Temperature & Salinity & Pressure) <br/>
`=ee.SEAWTR_DENSITY()`

Input as 25°C, 35 g/kg salinity, 1.013 bara <br/>
`=ee.SEAWTR_DENSITY(25, 35, 1.013)`     
`=1023.34` kg/m3 density of seawater  <br/>

ex) Get Seawater Specific Heat with (Temperature & Salinity & Pressure) <br/>
`=ee.SEAWTR_SPCHEAT(25, 35, 1.013)`     
`=3.99` kJ/kg-K specific heat  <br/>

**Typical Input Ranges:**
- Temperature: 0 to 120°C
- Salinity: 0 to 120 g/kg (seawater ~35 g/kg)
- Pressure: 0 to 12 bar.a

## Compressible Gas 
### GERG2008 - Gas Property
=ee.GASxxx()

ex) Get Denity with (Dry Temperature, Relative Humid Ratio and Ambient Pressure) <br/>
`=ee.GASRHO_PT()`

Input as 2 bara, 35 deg.C Temperature with FluidName<br/>
`=ee.GASRHO_PT(2, 35, Methane[0.6]&Ethane[0.4])`     
`=ee.GASRHO_PT(2, 35, B3_excel Range)`     
`=1.70111` kg/m3 density of Gas  <br/>

### ISO 6976 - Natural Gas Heating Value
=ee.GASxxx()

ex) Get LHV Lower Heating Value_kJ/kg (option with temperature) <br/>
`=ee.GASLHVMASS()`

Input with FluidName <br/>
`=ee.GASLHVMASS(Methane[0.6]&Ethane[0.4])`    
`=48630.14` kJ/kg as LHV of Gas <br/>

## Fluid Composition Helper
=ee.FluidString()

ex) Create fluid composition string with (fluid names & fractions & ratio method) <br/>
`=ee.FluidString()`

Input fluid names and mole/mass fractions to create composition string for gas calculations <br/>
`=ee.FluidString({Methane;Ethane;Propane}, {0.6;0.3;0.1}, "mol")`     
`=Methane[0.6]&Ethane[0.3]&Propane[0.1]` composition string  <br/>

Use this string directly in gas property functions <br/>
`=ee.GASRHO_PT(2, 35, FluidString({Methane;Ethane}, {0.6;0.4}))`     

**Note:** 
- Default ratio method is "mol" (mole basis)
- Can also use "mass" for mass basis
- Can reference Excel ranges for names and fractions
- Used with GERG2008 gas functions

## Friction Factor of pipe
=ee.FRICTIONFACTOR()

ex) Get friction factor with (dia_mm, roughness_mm, reynolds) <br/>
Input as DN200, 202.74 mm pipe inner diameter, 0.046 carbon pipe absolute roughness, reynolds <br/>
`=ee.FRICTIONFACTOR(202.74, 0.046, 213850)`     
`=0.0171` friction factor  <br/>

## Interpolation as Linear, Cubic
=ee.INTERPOLATION1D()

ex) Get interpolated value with (X value, X-Table, Y-Table, "Linear"=Option Default) <br/>
- linear method   
`=ee.INTERPOLATION1D(1.5, {1;2;3}, {10;20;50})`     
`=ee.INTERPOLATION1D(1.5, B1:B3_excel Range, C1:C3_excel Range)`     
`=15` get result  <br/>

- cubic method   
`=ee.INTERPOLATION1D(1.5, {1;2;3}, {10;20;50}, "cubic")`     
`=ee.INTERPOLATION1D(1.5, B1:B3_excel Range, C1:C3_excel Range, "cubic")`     
`=10` get result  <br/>
