Seawater는 MIT에서 개발한 해수 물성 계산 함수입니다.   
온도, 압력, 염도를 입력인수로 사용하여 각종 물성값을 제공합니다.

**Operating Ranges**
- Temperature: 0 to 180°C
- Pressure: 0 to 120 bar.a
- Salinity: 0 to 120 g/kg

**Seawater function UNIT**

import TableSeawater from './TableSeawater.astro';

_**Notes**_
- All functions accept temperature in Celsius, pressure in bar absolute, and salinity in grams per kilogram
- Reference states (T0, S0, P0) are required for flow exergy calculations
- Functions are based on the MIT seawater thermodynamic property library
- Properties are calculated using the IAPWS Industrial Formulation
