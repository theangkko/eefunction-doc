## 증기/물 - IF97 증기표
=ee.STEAMxxx()

예) 압력과 온도로 증기 엔탈피 구하기 <br/>
`=ee.STEAMH_PT()`

입력값: 압력 2 bara, 온도 150°C  <br/>
`=ee.STEAMH_PT(2, 150)`    
`=2769.09` kJ/kg 증기 엔탈피 <br/>

<br />


## 해수 특성 - MIT 모델
=ee.SEAWTR_xxx()

MIT 해수 특성 라이브러리 기반 해수 열물성.  
모든 함수는 온도(°C), 염도(g/kg), 압력(bar.a)을 입력값으로 사용합니다.

예) 온도, 염도, 압력으로 해수 밀도 구하기 <br/>
`=ee.SEAWTR_DENSITY()`

입력값: 온도 25°C, 염도 35 g/kg, 압력 1.013 bara <br/>
`=ee.SEAWTR_DENSITY(25, 35, 1.013)`     
`=1023.34` kg/m3 해수 밀도  <br/>

예) 온도, 염도, 압력으로 해수 비열 구하기 <br/>
`=ee.SEAWTR_SPCHEAT(25, 35, 1.013)`     
`=3.99` kJ/kg-K 비열  <br/>

**일반적인 입력 범위:**
- 온도: 0 ~ 120°C
- 염도: 0 ~ 120 g/kg (해수 ~35 g/kg)
- 압력: 0 ~ 12 bar.a

<br />

## 습공기 - ASHRAE
=ee.HUMIDAIR_xxx()

예) 건구온도, 상대습도, 대기압으로 밀도 구하기 <br/>
`=ee.HUMIDAIR_RHO_TDRY_RH_P()`

입력값: 온도 35°C, 상대습도 60%, 압력 1.013 bara <br/>
`=ee.HUMIDAIR_RHO_TDRY_RH_P(35, 0.6, 1.013)`     
`=1.13082` kg/m3 공기 밀도  <br/>

## 압축성 가스 
### GERG2008 - 가스 특성
=ee.GASxxx()

예) 압력, 온도, 유체명으로 밀도 구하기 <br/>
`=ee.GASRHO_PT()`

입력값: 압력 2 bara, 온도 35°C, 유체명<br/>
`=ee.GASRHO_PT(2, 35, Methane[0.6]&Ethane[0.4])`     
`=ee.GASRHO_PT(2, 35, B3:B3 (Excel 범위))`     
`=1.70111` kg/m3 가스 밀도  <br/>

### ISO 6976 - 천연가스 발열량
=ee.GASxxx()

예) 저위발열량(LHV) 구하기 (kJ/kg, 온도 옵션 포함) <br/>
`=ee.GASLHVMASS()`

입력값: 유체명 <br/>
`=ee.GASLHVMASS(Methane[0.6]&Ethane[0.4])`    
`=48630.14` kJ/kg 가스 LHV <br/>

## 유체 조성 도우미
=ee.FluidString()

예) 유체명, 분율, 비율 방식으로 유체 조성 문자열 생성 <br/>
`=ee.FluidString()`

유체명과 몰/질량 분율을 입력하여 가스 계산용 조성 문자열 생성 <br/>
`=ee.FluidString({Methane;Ethane;Propane}, {0.6;0.3;0.1}, "mol")`     
`=Methane[0.6]&Ethane[0.3]&Propane[0.1]` 조성 문자열  <br/>

가스 특성 함수에서 이 문자열을 직접 사용 <br/>
`=ee.GASRHO_PT(2, 35, FluidString({Methane;Ethane}, {0.6;0.4}))`     

**참고:** 
- 기본 비율 방식은 "mol" (몰 기준)
- 질량 기준은 "mass" 사용 가능
- 유체명과 분율에 Excel 범위(예: B1:B3) 참조 가능
- GERG2008 가스 함수와 함께 사용

## 배관 마찰계수
=ee.FRICTIONFACTOR()

예) 내경·조도·레이놀즈수로 마찰계수 구하기 <br/>
입력값: DN200, 내경 202.74 mm, 절대조도 0.046 (탄소강), 레이놀즈수 <br/>
`=ee.FRICTIONFACTOR(202.74, 0.046, 213850)`     
`=0.0171` 마찰계수  <br/>

## 선형 및 3차 보간
=ee.INTERPOLATION1D()

예) X값, X-테이블, Y-테이블로 보간값 구하기 ("Linear"=기본 옵션) <br/>
- 선형 방식   
`=ee.INTERPOLATION1D(1.5, {1;2;3}, {10;20;50})`     
`=ee.INTERPOLATION1D(1.5, B1:B3 (Excel 범위), C1:C3 (Excel 범위))`     
`=15` 결과값  <br/>

- 3차 보간 방식   
`=ee.INTERPOLATION1D(1.5, {1;2;3}, {10;20;50}, "cubic")`     
`=ee.INTERPOLATION1D(1.5, B1:B3 (Excel 범위), C1:C3 (Excel 범위), "cubic")`     
`=10` 결과값  <br/>
