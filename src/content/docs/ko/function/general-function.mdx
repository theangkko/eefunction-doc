이 페이지는 단위 변환, 물리 상수 및 일반적인 계산을 위한 유틸리티 함수를 제공합니다.

## 단위 변환
=ee.CONVERT()

예) 서로 다른 단위 간 값 변환 <br/>
`=ee.CONVERT(값, 원래_단위, 변환_단위)`

**온도 변환** <br/>
`=ee.CONVERT(100, "C", "F")` → `212` °F  <br/>
`=ee.CONVERT(273.15, "K", "C")` → `0` °C  <br/>

**압력 변환** <br/>
`=ee.CONVERT(1, "bar", "psi")` → `14.504` psi  <br/>
`=ee.CONVERT(101.325, "kPa", "atm")` → `1` atm  <br/>

**길이 변환** <br/>
`=ee.CONVERT(1, "m", "ft")` → `3.281` ft  <br/>
`=ee.CONVERT(25.4, "mm", "inch")` → `1` inch  <br/>

**질량 변환** <br/>
`=ee.CONVERT(1, "kg", "lb")` → `2.205` lb  <br/>
`=ee.CONVERT(1000, "kg", "ton")` → `1` ton  <br/>

**지원되는 단위 카테고리:**
- **온도**: C (섭씨), F (화씨), K (켈빈)
- **압력**: bar, psi, Pa, kPa, MPa, atm
- **길이**: m, cm, mm, km, ft, inch, yard
- **질량**: kg, g, mg, lb, oz, ton
- **부피**: m3, L, mL, gal, ft3
- **에너지**: J, kJ, cal, kcal, BTU, kWh

## 물리 상수
=ee.CONSTANT()

예) 보편적인 물리 상수 가져오기 <br/>
`=ee.CONSTANT(상수명)`

**기체 상수** <br/>
`=ee.CONSTANT("R")` → `8.314` J/(mol·K)  <br/>

**표준 중력 가속도** <br/>
`=ee.CONSTANT("g")` → `9.807` m/s²  <br/>

**아보가드로 수** <br/>
`=ee.CONSTANT("Na")` → `6.022e23` mol⁻¹  <br/>

**볼츠만 상수** <br/>
`=ee.CONSTANT("k")` → `1.381e-23` J/K  <br/>

**플랑크 상수** <br/>
`=ee.CONSTANT("h")` → `6.626e-34` J·s  <br/>

**광속** <br/>
`=ee.CONSTANT("c")` → `2.998e8` m/s  <br/>

**슈테판-볼츠만 상수** <br/>
`=ee.CONSTANT("sigma")` → `5.670e-8` W/(m²·K⁴)  <br/>

**사용 가능한 상수:**
- **R**: 기체 상수 (8.314 J/(mol·K))
- **g**: 표준 중력 가속도 (9.807 m/s²)
- **Na**: 아보가드로 수 (6.022e23 mol⁻¹)
- **k**: 볼츠만 상수 (1.381e-23 J/K)
- **h**: 플랑크 상수 (6.626e-34 J·s)
- **c**: 광속 (2.998e8 m/s)
- **sigma**: 슈테판-볼츠만 상수 (5.670e-8 W/(m²·K⁴))
- **pi**: 원주율 (3.14159...)
- **e**: 오일러 수 (2.71828...)
