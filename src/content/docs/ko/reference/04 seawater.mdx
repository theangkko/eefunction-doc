Seawater은 다음 코드를 사용합니다
- <a href="https://web.mit.edu/seawater/" target="_blank" rel="noopener noreferrer">MIT Seawater Properties</a> by MIT

Seawater는 MIT에서 개발한 해수 물성 계산 함수입니다.   
온도, 압력, 염도를 입력인수로 사용하여 각종 물성값을 제공합니다.

**동작 범위**
- 온도: 0 to 180°C
- 압력: 0 to 120 bar.a
- 염도: 0 to 120 g/kg

**Seawater 함수 단위**

import TableSeawater from '../../reference/TableSeawater.astro';

<TableSeawater />

_**참고**_
- 모든 함수는 온도(°C), 절대압(bar.a), 염도(g/kg)를 입력값으로 사용합니다.
- 유동 엑서지 계산에는 기준 상태(T0, S0, P0)가 필요합니다.
- 함수는 MIT 해수 열역학 물성 라이브러리를 기반으로 합니다.
- 물성값은 IAPWS Industrial Formulation을 사용해 계산합니다.
