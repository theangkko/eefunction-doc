설치하는 방법에 대해서 설명합니다. 크게 2가지 방법이 있습니다.
- MS AppSouce에서 추가하기
- Excel Add-ins 에서 검색하여 추가하기

`기존 제공하던, Side-Loading방식의 설치방법은 Update를 중단합니다.`

<br/>


### Using Microsoft AppSource

- MS AppSource URL 주소로 이동합니다.  <a href="https://appsource.microsoft.com/en-us/product/office/WA200007295?tab=Overview" target="_blank">LINK</a>
- `Get it Now`버튼을 눌러 Office Add-ins을 추가합니다.
- Office365(excel)에 Log-in 상태가 아니라면 Log-in 과정을 거칩니다.
- 정상적으로 설치가 되면, 상단 메뉴에 `eeFunction` 메뉴가 생성됩니다.

![Web-Excel](../../../assets/eefunction/Screenshot_appsource_url02.png)
![Web-Excel](../../../assets/eefunction/Screenshot_appsource_url03.png)

### For Desktop App-Excel, only some functions are available
기본적으로 Office Add-ins은 Web/Desktop 구분없이 사용가능합니다.    

eeFunction에서 제공하는 일부 함수는 Web Assembly 기반으로 작성되어 있습니다.   
이는 브라우저에서 동작하는 Web-Excel에서는 문제가 없지만 Desktop-Excel에서는 동작하지 않습니다.   
예를 들어, `=ee.PROPSSI`의 CoolProp 명령어가 이에 해당합니다.
