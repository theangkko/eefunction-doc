This page records updates and major changes.

<br/>



#### 2026-01
- Added Ammonia(NH3) Property functions entered with `=ee.NH3__()`.
- Added Hydrogen(H2) Property functions entered with `=ee.HYDROGEN__()`.

#### 2024-12
- Added Seawater Property functions entered with `=ee.SEAWTR__()`.
- Added Gas Heating Value related functions.

#### 2024-05
- Updated extensive content in eeFunction Docs.
- Organized function descriptions and unit lists for reference.

#### 2024-02
- Opened the eeFunction Docs site.


#### 2023-07
- Registered on MS AppSource, making installation easier.
- Previously installed using Local Side Loading method.


#### 2023-04
- Added Humid Air related functions using the `psychrolib` Library.


#### 2023-03
- Added IF97 Steam Table functions using the `xSteam` Library.
- Added Gas Property functions using the `GERG2008` Library.


#### 2022-05
- Created the initial version as an Office Add-in.
- Created to use CoolProp commands like `=ee.PROPSSI` in Web-Excel.
